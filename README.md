# Real-time Chat Application

A full-stack real-time chat application built with React.js, Node.js, and Socket.io.

## Features
- Real-time messaging using Socket.io
- Online users list with live updates
- User join/leave notifications
- Typing indicators
- Responsive UI built with Tailwind CSS

## Tech Stack
**Frontend:** React.js, Redux, Tailwind CSS, Socket.io-client  
**Backend:** Node.js, Express.js, Socket.io

## Getting Started

### Backend
```bash
cd chat-app/server
npm install
node index.js
```

### Frontend
```bash
cd chat-app/client
npm install
npm start
```

## Usage
1. Open http://localhost:3000 in your browser
2. Enter a username and click Join
3. Start chatting in real-time
4. Open multiple tabs to simulate multiple users
